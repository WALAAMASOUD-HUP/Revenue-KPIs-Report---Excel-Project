
Revenue KPIs Dashboard – Excel Project
This project showcases an interactive Revenue KPIs Dashboard built entirely in Excel using Pivot Tables, Pivot Charts, and slicers.
The dashboard provides a clear and dynamic view of key business metrics, helping users analyze revenue performance across multiple dimensions.
________________________________________
🚀 Project Overview
The dashboard highlights essential revenue insights, including:
•	Total Revenue
•	Quantity Sold
•	Average Revenue per Order
•	Revenue Breakdown by:
o	Category
o	Sub-Category
o	Ship Mode
o	Country
o	Order Year
Slicers are integrated for easy filtering, enabling the user to explore the data interactively.

This dashboard was built as a practical exercise to enhance data analysis and visualization skills using Excel.
The project demonstrates how Excel can deliver professional, interactive analytics without the need for advanced BI tools.
